# Touch Slider

Easy to use carousel, with infinite loop and lightweight, no dependencies.

# Installation
```
<script type="text/javascript" src='t_carusel.js'></script>
```
# Usage
```
  <div id="baner">
      <div>...</div>
      <div>...</div>
      <div>...</div>
  </div>
```
```
let baner = new Carusel('baner', {
  'touch': true,
  'dots': true,
  'arrows': true,
  'loop': true,
  'index': 1,
})
baner.start()
```
add css file with your styles
