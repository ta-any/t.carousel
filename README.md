# Touch Slider

Easy to use carousel, with infinite loop and lightweight, no dependencies.

# Installation
```
<script type="text/javascript" src='t_carusel.js'></script>
```
# Usage
```
  <div id="baner">
      <div>...</div>
      <div>...</div>
      <div>...</div>
  </div>
```
```
let baner = new Carusel('baner', {
  ...
})
baner.start()
```
